# 📅 StudySync

StudySync is a full-stack web application designed to help students manage academic stress by organizing exams, assignments, and study schedules in a structured and efficient way.

## 🚀 Features

- 🔐 User Authentication (Register/Login/Logout)
- 📝 Task Management (Add, view, and complete tasks)
- 📊 Dashboard with task overview
- ⏰ Deadline tracking with priority levels
- 🤝 Foundation for peer accountability features
- 📱 Clean and responsive UI using Bootstrap

## 🧠 Problem It Solves

Students often struggle with managing multiple subjects, assignments, and deadlines, leading to last-minute stress and poor productivity. StudySync provides a simple yet effective way to plan tasks, stay consistent, and reduce academic pressure.

## 🛠️ Tech Stack

- **Backend:** Python, Flask  
- **Frontend:** HTML, CSS, Bootstrap  
- **Database:** SQLite (SQLAlchemy ORM)  
- **Authentication:** Flask-Login  
- **Forms:** Flask-WTF  

## 💡 Future Improvements

- Smart auto-scheduler (AI-based task distribution)
- Calendar view for better planning
- Peer accountability groups
- Notifications and reminders
- Progress analytics and charts

## 🎯 Goal

To create a practical and easy-to-use tool that helps students stay organized, consistent, and stress-free during their academic journey.
